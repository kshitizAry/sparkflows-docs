
Quick Start Guide
=========

The quick start guide gets you started with Fire Insights. 

Let's get started! 

.. panels::
    :container: container-lg pb-3
    :column: col-lg-6 col-md-6 col-xs-12 p-2

    :doc:`/user-guide/quick-start/1-create-application`
    
    Create a project. It holds datasets, workflows, reports, dashboards, etc. Multiple groups can work on a project.

    ---

    :doc:`/user-guide/quick-start/2-upload-data-files`
    
    Upload your data files to be used in workflows.

    ---

    :doc:`/user-guide/quick-start/3-create-datasets`
    
    Create datasets on top of the data files. Datasets contain the schema of the data files and can be used in workflows.

    ---

    :doc:`/user-guide/quick-start/4-create-workflow`
    
    Create your workflow for processing the data, exploring it, building models, etc.
   
    ---

    :doc:`/user-guide/quick-start/5-execute-workflow`
    
    Finally execute the worflow.

    ---

    :doc:`/user-guide/quick-start/sample-workflows`
    
    Sample workflows.

    ---

    :doc:`/user-guide/quick-start/6-build-report`

    Create Report
