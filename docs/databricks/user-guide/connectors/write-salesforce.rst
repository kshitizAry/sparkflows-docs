Save Salesforce using Spark Connector
===============

Fire Insights enables you to write to Salesforce.

Save Salesforce
-----------

The below parameters are used when configuring this processor.

.. list-table::
      :widths: 5 10
      :header-rows: 1

      * - Name
        - Description
      * - USER NAME
        - USER NAME of Salesforce
      * - PASSWORD
        - PASSWORD of Salesforce
      * - SFOBJECT 
        - Salesforce Object
