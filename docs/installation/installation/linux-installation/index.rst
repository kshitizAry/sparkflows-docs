Linux Installation using TGZ file
^^^^^^^^^^^^^^^^^^^^^^^^^^^

Sparkflows can be installed to run natively on Linux machines using the binary bundled as a TAR. It can also be connected to a Spark cluster to submit jobs to cluster.
